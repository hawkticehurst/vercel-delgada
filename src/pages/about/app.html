This is a Delgada website. Delgada is a small, wickedly fast, Rust-based frontend web compiler for building slim UIs.

You can make your own by typing the following into your command line:

npx degit delgada-ui/template my-delgada-project cd my-delgada-project npm install npm start

Delgada is best used for building small (mostly) static websites.

You won't find any fancy diffing algorithms, virtual doms, reactive data, stateful components, or declarative syntax here. It's just plain vanilla HTML, CSS, and JavaScript that can be organized and structured in a way that is far more composable and easy to reason about.
